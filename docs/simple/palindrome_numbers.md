# Palindrome Numbers

- A number is palindrome if its reverse is same as number.
- Number is >=0
- A single digit number is always a palindrome number.

**Input:** n = 12321
**Output:** Yes

**Input:** n = 2332
**Output:** Yes

**Input:** n = 5
**Output:** Yes

**Input:** n = 34
**Output:** No

**Input:** n = 123
**Output:** No
